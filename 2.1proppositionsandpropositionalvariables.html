Propositions and Propositional Variables

Within the area of data science it is crucial to understand how to construct and utilise computer algorithms. An algorithm can be effectively understood as a series of rigorous procedures in which each result follows from the truth of a previous result. Therefore, it becomes apparent that in order to understand and construct algorithms one must first understand the underlying framework on which they are built. This framework is otherwise known as logic. In particular, this section will exclusively study the area of propositional logic. Here, the primary object of study are propositions, or more specifically propositional variables, and the logical operators that connect them. The clear starting point to this topic is then the definition of a proposition.

Definition 1.1: Statement/Proposition

A statement/proposition is a declarative sentence that must be either true or false.

Example 1.2: Statement/Proposition

The following are examples of propositions:

  1. $ 2 $ is an integer.
  2. Paris is the capital city of France.
  3. The sun is purple.

Note that the last sentence is still a statement, but is a false statement. The truth value of a particular statement or proposition has no bearing on whether it is a statement or not. A neat way of checking whether a sentence is a proposition is to put the words "Is is true that" in front of the sentence and ask if this question makes sense. For example, the sentence "$ 2 $ is an integer" is indeed a proposition as it makes sense to ask "Is it true that $ 2 $ is an integer?". An example of a sentence that is not a proposition would be "Pythagoras' Theorem". The question "Is it true that Pythagoras' Theorem?" makes no sense and hence the sentence is not a proposition. A slight subtlety to this rule would be when addressing sentences of the form $ x > 2 $. That is sentences that contain a variable. It does make sense to ask "Is it true that $ x > 2 $?", however the truth of this sentence depends on the value of $ x $. Given that a specific truth value cannot be assigned, it is therefore not a proposition. Sentences of this form will be dealt with in a later section on first order logic.

Definition 1.3: Propositional Variable

A propositional variable is a variable that takes values of either true or false.

A propositional variable is very similar to a numerical variable except that rather than taking values in the real numbers it takes propositions as its values. That is to say that it does not represent a specific proposition or statement but is simply a placeholder for any possible proposition. A capital $ P $ or $ Q $ will be used to denote a propositional variable similar to how $ x $ and $ y$ are used to denote numerical variables in algebra. Within the context of computing, a propositional variable may be understood as a function that only takes binary values. It is often the case that a true value is assigned to the number 1 and a false value is assigned the number 0. It is important to note that it is not the role of propositional logic to address whether a particular propositional variable is true or not. For the purposes of this course, propositional logic is simply a framework on which mathematical proofs will be constructed. Further, it may also be understood as the framework on which computer algorithms are built.
